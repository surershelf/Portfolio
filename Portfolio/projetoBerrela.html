Sistema de Cadastro e Gerenciamento Empresarial

Descrição do Projeto — Berrela Empreendimentos

O projeto desenvolvido para a empresa Berrela Empreendimentos surgiu da necessidade de melhorar o controle de clientes e o acompanhamento de pagamentos de parcelas de imóveis (apartamentos, casas e terrenos). Antes, a empresa realizava esse controle de maneira manual, o que dificultava a organização, aumentava o risco de erros e atrasava a tomada de decisões importantes, como o acompanhamento de inadimplência e previsão de recebimentos.

Para atender a essa necessidade, foi criado um sistema de gestão simples e funcional, utilizando a linguagem de programação Python e a biblioteca gráfica Tkinter para a construção da interface. A solução desenvolvida oferece um conjunto completo de funcionalidades de cadastro (CRUD), permitindo:

  • Adicionar, atualizar, excluir e visualizar registros de clientes;
  • Consultar clientes inadimplentes (que não realizaram o pagamento no mês vigente);
  • Visualizar os pagamentos que vencerão nos próximos 7 dias;
  • Gerenciar informações como nome, tipo de imóvel, telefone de contato, data de vencimento da parcela e valor da parcela.

O sistema apresenta uma interface intuitiva, com telas separadas para cada função e navegação simplificada através de botões. Cada ação importante é confirmada com mensagens visuais, garantindo segurança e clareza ao usuário.

Assim, o novo sistema trouxe maior agilidade, confiabilidade e praticidade para a rotina de gestão da Berrela Empreendimentos, otimizando o controle financeiro e fortalecendo o relacionamento com seus clientes.


Tela Inicial

A imagem apresenta a interface principal do sistema "Berrela Empreendimentos", que oferece um CRUD completo para gerenciamento de clientes e pagadores, permitindo adicionar, atualizar, excluir e visualizar cadastros. Também é possível consultar pagamentos em atraso e verificar os pagamentos previstos para os próximos 7 dias através do botão "Ver Pagamentos Próximos"


Tela de Cadastro

A imagem ilustra a tela de inserção de novos registros no sistema "Berrela Empreendimentos", pertencente ao módulo de cadastro (CRUD). O formulário permite o preenchimento dos seguintes campos obrigatórios: Nome, Tipo de Imóvel (apartamento, casa ou terreno), Data de Vencimento da Parcela, Telefone e Valor da Parcela.

Após o preenchimento, o usuário deve clicar no botão "Salvar" para armazenar as informações no banco de dados. Em caso de sucesso, o sistema exibe uma mensagem de confirmação indicando a inclusão do novo registro.


Tela de Atualização

A imagem ilustra a tela de atualização de registros no sistema "Berrela Empreendimentos", pertencente ao módulo de cadastro (CRUD). O usuário pode pesquisar um cliente existente pelo nome, utilizando o campo de busca e o botão "Pesquisar". Após localizar o registro, é possível editar os campos de Nome, Tipo de Imóvel (apartamento, casa ou terreno), Data de Vencimento da Parcela, Telefone e Valor da Parcela.

Após realizar as alterações necessárias, o usuário deve clicar no botão "Salvar" para atualizar as informações no banco de dados.


Tela de Leitura

A imagem ilustra a tela de visualização de cadastros no sistema "Berrela Empreendimentos", pertencente ao módulo de consulta (CRUD). A interface exibe uma lista de clientes cadastrados, apresentando as informações dos clientes.
Os registros são separados por linhas tracejadas para facilitar a leitura.


Tela de Exclusão

A imagem ilustra a tela de exclusão de registros no sistema "Berrela Empreendimentos", pertencente ao módulo de cadastro (CRUD). O usuário pode informar o nome do cliente a ser excluído e, em seguida, clicar no botão "Pesquisar" para localizar e excluir o registro correspondente. Após a exclusão, o sistema exibe uma mensagem de confirmação indicando que o registro foi removido com sucesso.


Tela de Pagamentos Atrasados

A imagem ilustra a tela de visualização de clientes inadimplentes no sistema "Berrela Empreendimentos". A interface exibe a lista de pessoas que não realizaram o pagamento da parcela no mês atual, apresentando informações dos clientes.

Cada registro é separado por uma linha tracejada para facilitar a leitura.


Próximos Pagamentos
