Snake Game

Projeto inspirado no clássico jogo da cobrinha, desenvolvido com o objetivo de aprimorar a lógica de programação e aprofundar o domínio da linguagem Python de uma forma divertida. A proposta envolveu trabalhar com estruturas de controle, manipulação de gráficos e desenvolvimento de uma lógica de jogo fluida e funcional.
