Snake Game – Python

Projeto inspirado no clássico jogo da cobrinha, desenvolvido com o objetivo de aprimorar a lógica de programação e aprofundar o domínio da linguagem Python de uma forma divertida e prática.

Tecnologias Utilizadas:

Python: linguagem base para o desenvolvimento de toda a lógica do jogo, controle dos movimentos, detecção de colisões e pontuação.

Biblioteca gráfica Pygame : utilizada para renderizar os elementos visuais, como a cobrinha, a comida e o campo de jogo, garantindo uma experiência visual simples e interativa.

Funcionalidades:

Controle da cobrinha por meio das setas direcionais do teclado.
Crescimento progressivo da cobrinha ao consumir os itens no campo. Sistema de pontuação com aumento de dificuldade conforme o jogo avança. Detecção de colisões com o próprio corpo, encerrando o jogo ao atingir a condição de falha.

Objetivo:

Praticar conceitos fundamentais de lógica de programação, como estruturas condicionais e laços de repetição, além de desenvolver habilidades em manipulação gráfica e construção de jogos simples com Python.
