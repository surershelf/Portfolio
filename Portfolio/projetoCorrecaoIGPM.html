Correção IGPM

Aplicativo de Correção de Aluguel pelo IGPM – Python + Tkinter

Este projeto foi desenvolvido especialmente para atender às necessidades de uma imobiliária, com o objetivo de automatizar o processo de correção de valores de aluguel com base no índice IGPM (Índice Geral de Preços do Mercado).

Tecnologias Utilizadas:

Python: responsável por toda a lógica do sistema, incluindo o cálculo da correção com base no índice informado.

Tkinter: biblioteca padrão do Python utilizada para construir a interface gráfica (GUI), garantindo praticidade e facilidade de uso para os colaboradores da imobiliária.

Funcionalidades:

Interface intuitiva para entrada de dados como valor atual do aluguel e índice IGPM acumulado.
Cálculo automático do novo valor corrigido.
Resultado exibido em tempo real com clareza, evitando erros de digitação ou cálculos manuais.
Design simples, direto e funcional, pensado para o ambiente de trabalho.

Objetivo:

Facilitar e agilizar o trabalho da equipe da imobiliária, oferecendo uma ferramenta leve e eficiente para atualização de contratos de locação com base no IGPM.
