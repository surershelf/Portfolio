Jogo Godot

Jogo 2D Criado com Godot – Projeto Educacional

Este projeto consiste em um jogo simples em 2D desenvolvido com a engine Godot, seguindo a documentação oficial disponível no site da Godot Engine. Ele foi criado com o intuito de aprimorar habilidades em desenvolvimento de jogos de forma divertida e prática.

Tecnologias Utilizadas:

Godot Engine: responsável por toda a lógica, animação e funcionamento do jogo, utilizando os recursos oferecidos pela plataforma para construção de jogos em 2D.

Assets "Dodge the Creeps": utilizados conforme o tutorial oficial "Seu Primeiro Jogo 2D", fornecendo base visual e interativa ao projeto.

Funcionalidades:

Projeto interativo com movimentação e colisão entre elementos do jogo.
Introdução prática aos conceitos de game loop, sprites, sinais e cenas no Godot.
Estrutura clara e organizada, ideal para iniciantes no desenvolvimento de jogos.
Possibilidade de expansão e customização, servindo como base para projetos mais avançados.

Objetivo:

Servir como ferramenta de aprendizado para quem deseja ingressar no universo de desenvolvimento de jogos, oferecendo um exemplo prático e acessível utilizando uma das engines mais populares do mercado.
